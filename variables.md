## Ejercicios de la clase del 13 de enero 

*Entradas*:

```
a = 5
b = 18
c = 15
d = 25
```

*Calculos*:

```
a = 5 + 10 = 15
b = 18 + 5 - 15 = 8
c = 15 + 4 + 8 = 27
d = 25 + 8 + 15 = 38
a = 15 + 1 = 16
b = 8 + 27 = 35
c = 35 + 27 = 62
d = 35 + 35 = 70 
```
*Salidas:*
```
a = 16
b = 35
c = 62
d = 70 
```
